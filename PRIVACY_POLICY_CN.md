# 隐私政策

最后更新：2026-03-17

## 概述

Memex（以下简称"本应用"）是一款本地优先的个人知识管理应用。我们致力于保护您的隐私。本政策说明本应用如何处理您的数据。

## 数据收集

Memex 采用本地优先的设计。您在应用中创建的所有数据（包括文字、照片、语音录音和 AI 生成的内容）默认存储在您的设备上。

Memex 不运营自有服务器，不会在我们控制的任何服务器上收集或存储您的个人数据。您的数据离开设备的唯一情况是您主动配置并使用第三方 LLM 服务商（详见下方"第三方服务"章节）。

## 第三方服务

Memex 仅在您主动配置 API Key 后才会连接第三方大语言模型（LLM）服务商，包括：

- Google Gemini
- OpenAI
- Anthropic Claude
- AWS Bedrock

使用这些服务时，您的输入内容会发送至您选择的服务商以生成 AI 回复。这是数据离开您设备的唯一场景。关于各服务商如何处理您的数据，请参阅其隐私政策：

- [Google 隐私政策](https://policies.google.com/privacy)
- [OpenAI 隐私政策](https://openai.com/privacy)
- [Anthropic 隐私政策](https://www.anthropic.com/privacy)
- [AWS 隐私政策](https://aws.amazon.com/privacy/)

## 设备权限

Memex 可能会请求以下设备权限。通过这些权限获取的所有数据均在您的设备本地处理和存储，不会上传到我们的服务器。

- **相机** — 用于拍照记录。
- **麦克风** — 用于语音录入。
- **相册/媒体** — 用于选择设备中已有的照片用于记录。
- **位置** — 用于读取照片中嵌入的 GPS 元数据以支持地点相关的记录，以及在编辑记录时允许您在地图上选取位置。Memex 不会在后台追踪您的位置。位置数据仅存储在本地，不会向外部传输。
- **日历** — 用于在 AI 代理从您的输入中识别出日程安排意图时，向设备系统日历写入日历事件。Memex 不会读取您的系统日历。
- **提醒事项（iOS）** — 用于在 AI 代理从您的输入中识别出提醒意图时，在您的设备上创建提醒事项。Memex 不会读取您现有的提醒事项。
- **健康/健身（Android 上的 Health Connect，iOS 上的 HealthKit）** — 用于在您启用该功能时读取健康和健身数据。具体数据类型包括：步数、心率、静息心率、血压（收缩压和舒张压）、血氧、血糖、睡眠、活动消耗热量、体重和运动/锻炼记录。所有健康数据仅存储在本地，不会向外部传输。
- **活动识别** — 用于在 Health Connect / HealthKit 数据不可用时，通过设备计步器进行步数统计。
- **生物识别（Face ID / Touch ID / 指纹）** — 用于应用锁认证。生物识别数据完全由操作系统处理，Memex 不会访问或存储您的生物识别数据。
- **通知** — 用于发送本地提醒和后台任务状态更新。不涉及推送通知服务或外部服务器。
- **存储/文件** — 用于在您的设备上存储本地数据库、媒体文件和导出数据。在 Android 11 及以上版本中，如果您在"设置 > 数据存储"中选择了自定义工作区文件夹，本应用会请求"所有文件访问"（MANAGE_EXTERNAL_STORAGE）权限，以便在您选择的文件夹中读写工作区文件（时间线卡片、知识库、媒体资产等）。此权限仅在您主动选择自定义存储位置时才会请求。如果您使用默认应用存储，则不需要此权限。
- **网络** — 仅用于与您配置的第三方 LLM 服务商通信。不会向我们运营的任何服务器发送数据。

## 端侧处理

以下功能完全在您的设备上运行，不会向外部发送数据：

- OCR 文字识别（Google ML Kit，端侧运行）
- 图像标签与场景识别（Google ML Kit，端侧运行）
- 照片 EXIF 元数据提取（包括 GPS 位置信息）
- 本地数据库存储（基于 Drift 的 SQLite）
- 设备计步器步数统计

## 分析与追踪

Memex 不包含任何分析、追踪或广告 SDK。

## 数据保留与删除

您的所有数据均存储在您的设备本地。Memex 不会在外部服务器上保留任何数据。

- **保留期限**：只要您保持应用安装且未主动删除，数据将一直保留在您的设备上。由于没有数据被传输或存储到我们的服务器，因此不存在服务器端的保留期限。
- **删除单条记录**：您可以随时在应用内直接删除任何记录（文字、照片、语音录音、卡片、知识条目等）。删除后的记录将从本地数据库中永久移除，无法恢复。
- **删除全部数据**：您可以随时通过卸载 Memex 或在系统设置中清除应用数据来移除所有应用数据。这将永久删除所有本地存储的数据，包括数据库、文件和缓存内容。
- **第三方 LLM 服务商**：在 AI 交互过程中发送给第三方 LLM 服务商的数据，受该服务商自身的数据保留和删除政策约束。请参阅相应服务商的隐私政策（链接见上文）了解如何请求删除其可能处理过的数据。

## 儿童隐私

Memex 不会故意收集 13 岁以下儿童的任何信息。
